# Linear

A clean technical foundation for a production Next.js application.

## Stack

- Next.js App Router
- TypeScript
- React
- Tailwind CSS
- Framer Motion
- ESLint
- Prettier

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run typecheck
npm run format:check
```

## Source Layout

```text
src/
  app/          App Router routes, layouts, and global styles
  components/   Reusable UI, layout, and motion primitives
  config/       Application configuration
  lib/          Shared utilities
  styles/       Shared styling entry points
  types/        Shared TypeScript types
```
